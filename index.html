Virtual Pet

The Virtual Pet project was my first experience in using multiple classes I had created. The beginning of the game awards the player a hatchling phoenix, which they get to name. The player is then prompted with different options to care for their phoenix. If the phoenix is not properly cared for, or the player decides to quit, the phoenix will die and await the player's turn for rebirth.

Skills Used: Java - Connecting classes using a single package and importing Scanner to utilize user input.

Virtual Pet Shelter

Building upon the design of Virtual Pet is Virtual Pet Shelter. Rather than a phoenix, you have four pets in the shelter that caters to the black market crowd. You are able to care for the pets, adopt a pet, or add a pet to the shelter. Be warned, these are not your average, happy-go-lucky pets.

Skills Used: Java - Parent and child classes, refactoring, TDD
